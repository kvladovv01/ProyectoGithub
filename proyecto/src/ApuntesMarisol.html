1. HTML (HyperText Markup Language)

Función principal: Estructura el contenido de la web.

Etiquetas comunes:

  • <!DOCTYPE html>: Declaración que indica el tipo de documento.
  • <html>: Contenedor de todo el contenido de la página.
  • <head>: Incluye metadatos como <title>, <meta>, y enlaces a CSS.
  • <body>: Contiene el contenido visible de la página.
  • <h1> a <h6>: Encabezados de mayor a menor importancia.
  • <p>: Párrafos de texto.
  • <a href="URL">: Enlaces.
  • <img src="URL" alt="descripción">: Imágenes.

2. CSS (Cascading Style Sheets)

Función principal: Estiliza y da formato a las páginas web.

Selectores básicos:

  • elemento { propiedad: valor; }: Aplica estilos a todos los elementos de un tipo específico.
  • .clase { ... }: Selecciona elementos con una clase específica.
  • #id { ... }: Selecciona un único elemento con un ID específico.
  • elemento1 elemento2 { ... }: Selecciona elementos hijos.

Propiedades comunes:

  • color: Color del texto.
  • background-color: Color de fondo.
  • margin y padding: Espaciado exterior e interior.
  • border: Bordes (ancho, tipo y color).

Media queries: Adaptan los estilos a diferentes tamaños de pantalla.

        @media (max-width: 600px) {
            body {
                background-color: lightblue;
            }
        }

3. JavaScript

Función principal: Añade interactividad y funcionalidad dinámica a la web.

Conceptos clave:

  • Variables: Contenedores de datos. let, const, var.
  • Funciones: Bloques de código que realizan una tarea específica.
        function saludo(nombre) {
            console.log("Hola, " + nombre);
        }
        saludo("Marta");

Eventos: Respuestas a acciones del usuario (clics, teclado, etc.).

        document.getElementById("boton").onclick = function() {
            alert("Botón presionado");
        };

Manipulación del DOM: Permite modificar elementos HTML desde JavaScript.

        document.querySelector("h1").textContent = "Nuevo Título";
